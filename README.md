# Finance Tracker

A simple Python project to track income and expenses and visualize financial data.

## Features

* Add income and expenses
* Calculate total balance
* Visualize expenses using Matplotlib

## Technologies Used

* Python
* Pandas
* Matplotlib

## Installation

```bash
pip install pandas matplotlib
```

## Run the Project

```bash
python main.py
```
